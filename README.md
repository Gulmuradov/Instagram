In this project I have tried to create a clone of the Instagram app

# Installation 

```
  npx create-react-app .
```

## Run Locally

Go to the project directory

```bash
  cd instagram-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
